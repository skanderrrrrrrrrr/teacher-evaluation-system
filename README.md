# Teacher Evaluation System

## Description

This project is a web-based teacher evaluation system that allows students to evaluate their teachers through an online platform. The system helps collect feedback efficiently and provides administrators with evaluation reports.

## Features

* Student authentication
* Teacher evaluation forms
* Evaluation management
* Results and reports
* User-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL

## Installation

1. Download the project files.
2. Import the database into MySQL.
3. Configure the database connection.
4. Run the project on a local server XAMPP.
